# Onboarding step screenshots

Place PNG files here so the **Onboarding → Pages** charts can show a ~200px-tall preview under each step.

## Naming

URL pattern: `/screens/<page_name>.png` (same `page_name` as in analytics / app).

### Onboarding v2 (default flow)

- `hello.png`, `name.png`, `1.png`, `2.png`
- `summaryConclusion.png`, `summaryConclusionQuestion.png`, `summaryAI.png`
- `ps2.png`, `1.2.png`, `ps1.png`, `noPremium1.png`, `notification.png`, `tasks.png`

- `noPremium1.png`, `noPremium2.png`, `practice.png`, `notification.png`, `ps2.png`
